# 移除通用示例固定故事污染

## CLI 行为

- `interview` 的通用问题示例不再默认带入特定故事的专名、题材设定或固定势力。
- 新用户以非相关题材开始访谈时，输出仍保留示例分支，但示例只承担格式引导。

## 模板契约

- clarification core/cozy 模板中的示例改为题材无关描述，避免把旧示例误读为当前故事候选。
- 共创入口卡的默认示例同步泛化，不能把专属世界观词汇当作通用入口。

## 生成产物

- `clarifications.md` 和 plan preview 的未确认候选不应出现与当前 premise 无关的固定故事词。
- 未确认示例仍不会进入正典，且阅读层级应弱于用户已确认内容。

## 验证

- 新增“星尘驿站”回归测试，确保访谈与计划预览不出现旧示例污染词。
